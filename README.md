## CLI
Start bun
```bash
bun init
bun create vue
bun create next-app
bun create vite
```

To install dependencies:
```bash
bun install
```

To run:
```bash
bun run index.ts
bun run .
```

Express & Elysia:
```bash
bun install express
bun add -d @types/express
bun install elysia
```

React & React-dom:
```bash
bun add react react-dom
bun add -d @types/react-dom @types/react-dom
```# practice-bun
